# Перечень изменений ProtoEMU 1.0

## 1.0.42 от 07-04-2026

- Для SL версии в окне "О программе" отображается соответствующая ссылка для загрузки дистрибутива.
- Доработка кнопки "пуск/останов" - теперь корректно отображает состонияе при прерывании работы пульта.
- По умолчанию включен следящий режим для окна сообщений.
- Упрощена логика управления отображеним окон сообщений, наблюдения, ошибок.
- Реализован подъем окна сообщений при сохранении сообщений уровня FATAL.

## 1.0.41 от 25-03-2026

- Реализована возможность множественных подключений к Modbus-TCP серверу.
- Реализовано автосохранение текста программ в узел.
- Реализованы операторы +=, -= для типа Value.
- Восстановлена работа функции перетаскивания узлов из дерева проекта на форму.

## 1.0.40 от 09-02-2026

- Реализовано отображение в центральном виджете подсказки по примерам.
- Исправлен возможный крах при открытии документа.
- Исправлено корректное формирование значений типа Value из строк, float в свойствах виджетов.
- Реализован импорт транспортов Modbus-TCP из проектов AgavaSCADA/AgavaPLC.
